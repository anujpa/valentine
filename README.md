## Customization

Want to personalize it? Here are some easy customizations you can make:

### Change the Love Message

Find this section in the code and edit the text:

```html

    I knew you'd say yes! üíï<br><br>
    You've just made me the happiest person alive!<br>
    Can't wait to spend this Valentine's Day with you.<br><br>
    You're absolutely wonderful! üåπ
</div>
```

### Change the Colors

Modify the background gradient in the CSS:

```css
background: linear-gradient(135deg, #ff9a9e 0%, #fad0c4 50%, #ffd1ff 100%);
```

### Change the Question

Edit the main question text:

```html
<div class="question">Will you be my Valentine?</div>
```
